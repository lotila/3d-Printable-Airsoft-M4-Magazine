3d printable m4 airsoft magazine

### Feaures

- Capacity 39 BBs 

- Durable design

- Lightweight at 64g

- Low cost per magazine

- Full magazine indicator

### Requirements

- Blender 2.9

### Build

- Open magazine.blend file in blender.

- Move to scripting mode.

- Run script (takes around one minute to build)
